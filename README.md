 A Machine Learning Approach to Scholastic Performance Classification
 Overview
This project is a final semester academic project that predicts and categorizes student academic performance based on behavioral and academic input features such as grades, attendance, and study patterns. It uses multiple machine learning algorithms to identify the best-performing model.
Tech Stack
- Python  
- Scikit-learn  
- Pandas  
- NumPy  
- Matplotlib

Work Done
- Designed and implemented a classification pipeline to categorize student performance  
- Performed data preprocessing including cleaning and handling missing values  
- Trained and compared multiple machine learning algorithms  
- Evaluated models using accuracy, precision, recall, and F1-score  
- Selected the best-performing model based on evaluation metrics  
Results
- Best Model: Decision Tree  
- Accuracy: 89.69%  
- Gradient Boosting: 86.64%  
- SVM: 69.27%  
Algorithms Used
- Decision Tree  
- Random Forest  
- Support Vector Machine (SVM)  
- Gradient Boosting  
Conclusion
This project demonstrates the application of machine learning techniques to analyze and classify student performance, helping in identifying academic patterns and supporting data-driven educational insights.
